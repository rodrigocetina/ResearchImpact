## How to publish on GitHub Pages

1. Create a new GitHub repository (e.g. `research-matrix`)
2. Drag **`index.html`** into the repository (you can ignore this README if you prefer)
3. Go to **Settings → Pages**
4. Under *Branch*, select **main** and click **Save**
5. GitHub will give you a live URL like:
   `https://your-username.github.io/research-matrix/`

The page will be live within ~1 minute. That URL can be shared with anyone.

---

> The original `research-excellence-matrix.html` file is preserved separately and is identical in content to `index.html`.
